Como funciona?
  1. O sistema verifica se o PDF já existe (detecção por hash SHA-256)
  2. Se for duplicata, reutiliza o conteúdo já processado
  3. Se for novo, extrai o texto automaticamente
  4. Você será redirecionado para a página de geração de questões
  5. Escolha o nível de dificuldade e quantidade de questões desejadas
  6. As questões serão geradas usando IA no padrão IDECAN
